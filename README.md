
# HdxSpider
HdxSpider是用于爬取“好大学在线“课程爬虫

# 项目目的
在于方便同学们更好地利用免费地学习资源，节省在线缓冲时间

# 使用方法
1.下载并使用go1.13编译运行本程序

2.输入用户名和密码，即可爬取该账号下地课程

3.课程保存目录为当前程序运行路径下Data目录
